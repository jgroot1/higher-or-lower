# higher-or-lower

A simple higher or lower game i made in python. it is the first project i made in python so don't expect it to be good it works at least.

## How it works
The player can insert a number and then they can play a game of higher or lower with their number being the max number with a random generated number.
Every gues the player will get told if the number is higer or lower then their gues until they gues the right number.
When the user guesses the right number they will get told how many guesses it took and if they want to play again.

## Features

user can pick the highest number for the game themself

feedback on input

the user is notified when an invalid input is enterd, such as letters instead of numbers
